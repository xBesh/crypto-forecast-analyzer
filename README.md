# Crypto Forecast Analyzer

A comprehensive cryptocurrency market analysis and forecasting dashboard built with React, TypeScript, and Vite.

## Features

- **Real-time Dashboard**: Monitor cryptocurrency market data with live updates
- **Technical Analysis**: View detailed technical indicators and market trends
- **AI-Powered Forecasts**: Access machine learning-based price predictions
- **Portfolio Tracking**: Manage and track your cryptocurrency investments
- **Interactive Charts**: Visualize data with responsive, interactive charts
- **Market Sentiment**: Analyze market sentiment from multiple sources

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router DOM
- Recharts (Data Visualization)
- Lucide React (Icons)
